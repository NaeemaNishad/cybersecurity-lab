# 🌐 Networking Notes

A concise overview of cybersecurity fundamentals and core networking concepts.

---

## 🔐 Cybersecurity Basics

### 🛡️ CIA Triad
The CIA Triad forms the foundation of cybersecurity:

- **Confidentiality** → Ensures sensitive data is accessed only by authorized users (e.g., passwords, encryption)  
- **Integrity** → Ensures data remains accurate and unaltered  
- **Availability** → Ensures systems and data are accessible when needed  

---

### ⚠️ Types of Cyber Threats

- **Phishing** → Tricks users into revealing sensitive information  
- **Malware** → Malicious software (viruses, worms, trojans)  
- **DDoS (Distributed Denial of Service)** → Overloads a system to make it unavailable  
- **SQL Injection** → Injects malicious queries into databases  
- **Brute Force Attack** → Attempts multiple password combinations  
- **Ransomware** → Encrypts data and demands payment  

---

### 🎯 Attack Vectors

- **Social Engineering** → Manipulating users to gain information  
- **Wireless Attacks** → Exploiting insecure Wi-Fi networks  
- **Insider Threats** → Misuse of access by trusted individuals  

---

## 🌐 Networking Basics

### 📡 OSI Model
The OSI (Open Systems Interconnection) Model explains how systems communicate over a network through seven layers.

**Layers:**
1. **Physical** → Transmits raw bits  
2. **Data Link** → Ensures error-free transfer  
3. **Network** → Handles routing using IP addresses  
4. **Transport** → Provides reliable data delivery (TCP/UDP)  
5. **Session** → Manages communication sessions  
6. **Presentation** → Formats and encrypts data  
7. **Application** → Provides services to end users  

---

### 🔗 TCP/IP Protocol Suite

- **TCP (Transmission Control Protocol)** → Reliable, connection-oriented communication  
- **UDP (User Datagram Protocol)** → Fast, connectionless communication  

---

### 🌍 DNS (Domain Name System)
DNS translates human-readable domain names (e.g., google.com) into IP addresses for communication between devices.

---

### 🔒 HTTP vs HTTPS

- **HTTP (HyperText Transfer Protocol)** → Transfers data in plain text (not secure)  
- **HTTPS (HyperText Transfer Protocol Secure)** → Uses SSL/TLS encryption to secure communication  

---

### 🧭 IP Addressing

An IP address uniquely identifies a device on a network.

**Types:**
- **IPv4** (e.g., 192.168.1.1)  
- **IPv6** (longer, modern format)  

**Categories:**
- **Private IP** → Used within local networks  
- **Public IP** → Used on the internet  

---

### 🔀 Subnetting

Subnetting divides a large network into smaller subnets.

**Benefits:**
- Improved performance  
- Enhanced security  
- Efficient traffic management  

Uses a subnet mask (e.g., 255.255.255.0) to define network boundaries.

---

### 🔄 NAT (Network Address Translation)

NAT maps private IP addresses to a public IP address.

**Benefits:**
- Conserves IP addresses  
- Adds security by hiding internal network details
