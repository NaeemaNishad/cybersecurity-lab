# 🔐 Cybersecurity Lab Project

## 📌 Objective
To build strong fundamentals in cybersecurity, networking, cryptography, and perform hands-on testing using industry tools.

---

## 🛠️ Tools Used
- Kali Linux  
- Metasploitable2  
- Wireshark  
- Nmap  
- Burp Suite  
- Netcat  

---

## 🧠 Concepts Covered
- CIA Triad  
- Types of Cyber Attacks  
- OSI Model  
- TCP/IP Protocol  
- DNS, HTTP/HTTPS  
- IP Addressing, Subnetting, NAT  
- Cryptography Basics
